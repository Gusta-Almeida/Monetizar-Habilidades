# Projeto: Identificação de Habilidades  
### Mapeamento, organização e monetização de competências pessoais

---

## Visão Geral

Este repositório documenta o **Projeto: Identificação de Habilidades**, cujo objetivo é mapear e analisar competências pessoais, transformando-as em oportunidades reais de monetização.  
O processo combina **autoavaliação guiada**, **análise de mercado** e **uso estratégico de IA** para estruturar um inventário de habilidades aplicável a novas carreiras, serviços e produtos.

---

## Objetivos do Projeto

- Identificar habilidades técnicas, comportamentais e adquiridas pela experiência.  
- Relacionar cada habilidade a oportunidades de monetização.  
- Realizar pesquisa de mercado para analisar oferta, demanda e tendências.  
- Criar prompts estruturados para aprofundar análises futuras.  
- Documentar todo o processo de forma clara e versionada.

---

## Processo de Desenvolvimento

1. **Inventário Inicial:** levantamento de habilidades pessoais com base em histórico profissional e afinidades.  
2. **Classificação das Habilidades:** organização em categorias (técnicas, interpessoais, criativas etc.).  
3. **Análise de Monetização:** conexão entre habilidade e valor gerado para o mercado.  
4. **Pesquisa de Mercado:** identificação de serviços em alta e demandas emergentes.  
5. **Refinamento com IA:** uso de prompts estruturados para ampliar insights e análises.
